## ðŸ’Ž Pro Tips

1. **Layer your prompts**: Combine 5-7 different elements
2. **Be specific**: "dramatic rim lighting" > "good lighting"
3. **Add movement**: Static scenes are less engaging
4. **Include atmosphere**: Weather, particles, environmental effects
5. **Quality tags**: Always add quality enhancers at the end
6. **Consistency**: Use similar lighting/style across scenes in same story
7. **Pacing**: Vary intensity - build tension, release, repeat
